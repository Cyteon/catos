## Running it

#### Prerequisites
1. [Qemu](https://www.qemu.org/)
2. [Rust](https://www.rust-lang.org/)

Clone the project

```bash
  git clone https://github.com/Cyteon/catos
```

Go to the project directory

```bash
  cd catos
```

Run the OS in qemu

```bash
  cargo run
```

Run the tests

```bash
  cargo test
```
